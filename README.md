# 🚀 Emciar

Aplikasi berbasis Next.js yang dikembangkan sebagai proyek untuk menyediakan platform digital yang responsif dan modern.

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
